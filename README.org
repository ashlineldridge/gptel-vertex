* gptel-vertex
This package adds [[https://cloud.google.com/vertex-ai/docs][Google Cloud Vertex AI]] support to the Emacs [[https://github.com/karthink/gptel][gptel]] package.
** Installation
#+begin_src emacs-lisp
  (require 'gptel-vertex)
  (setq gptel-model 'claude-sonnet-4-5@20250929
        gptel-backend (gptel-make-vertex "VertexAI"
                                         :project-id "my-gcp-project"
                                         :location "us-west1"))
#+end_src
