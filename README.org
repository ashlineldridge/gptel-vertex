* gptel-vertex

This package adds Google Cloud Vertex AI support to the Emacs gptel package.

** Installation
#+begin_src emacs-lisp
  (require 'gptel-vertex)
  (setq gptel-model 'claude-sonnet-4-5@20250929
        gptel-backend (gptel-make-vertex "VertexAI"
                                         :project-id "my-gcp-project"
                                         :location "us-west1"))
#+end_src
